# TikTok Plus

为 TikTok 网页版添加一组顺手的键盘快捷键，让刷视频、互动和搜索更轻松。

## 功能

### 播放控制

| 快捷键 | 功能 |
| --- | --- |
| `Space` | 暂停 / 播放 |
| `↑` / `↓` | 上下切换视频 |
| `←` / `→` | 快退 / 快进 5 秒 |
| `H` | 播放器全屏 |
| `Esc` | 退出全屏 / 关闭快捷键列表 |

### 互动操作

| 快捷键 | 功能 |
| --- | --- |
| `Z` | 点赞 |
| `X` | 打开评论区 |
| `C` | 收藏 |
| `V` | 复制当前视频口令 |
| `B` | 开关弹幕 |
| `R` | 不感兴趣 |
| `G` | 关注作者 |

### 搜索与帮助

| 快捷键 | 功能 |
| --- | --- |
| `Shift + F` | 聚焦搜索框 |
| `Shift + ?` | 显示 / 关闭快捷键列表 |

## 安装

推荐使用用户脚本管理器安装：

- [Tampermonkey](https://www.tampermonkey.net/)
- [Violentmonkey](https://violentmonkey.github.io/)

安装管理器后，将脚本添加到管理器中并打开 `https://www.tiktok.com/` 即可使用。

## 注意事项

- 只在 `https://www.tiktok.com/*` 生效。
- 在输入框、搜索框、评论框中打字时，脚本会自动让出大部分快捷键，避免误触。
- TikTok 网页结构可能更新，若某个互动按钮失效，可以刷新页面或等待脚本更新。
